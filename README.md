# CODSOFT
   # 1. Number Guessing Game

A console-based C++ game where the player guesses a randomly generated number between 1 and 100.

## Features
- Random number generation each round
- Hints (Too High / Too Low) after each guess
- Performance feedback based on number of attempts
- Option to replay

## How to Run
g++ number_guessing.cpp -o number_guessing
./number_guessing

# 2. Simple Calculator

A console-based C++ calculator supporting basic arithmetic operations.

## Features
- Addition, Subtraction, Multiplication, Division
- Division by zero handling
- Invalid input handling with error messages

## How to Run
g++ calculator.cpp -o calculator
./calculator

# 3. Tic Tac Toe

A two-player console-based Tic Tac Toe game in C++ with colored output and score tracking.

## Features
- Custom player names
- Color-coded X and O markers
- Win, draw detection
- Score tracking across multiple rounds

## How to Run
g++ tictactoe.cpp -o tictactoe
./tictactoe
